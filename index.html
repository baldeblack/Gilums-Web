Desarrollo Web

  • SITIOS AUTO-GESTIONABLES
  • El cliente podrá administrar su sitio web de forma sencilla, gestionar todo el contenido dinámicamente y en un entorno amigable. Desarrollamos sitios web interactivos, portales, blogs, e-commerce, integrados con redes sociales, con gestión de usuarios de acceso a los paneles de administración. En nuestros proyectos nos encargamos del desarrollo en todas sus fases, desde el diseño web hasta la puesta a punto del sitio.
  • APLICACIONES WEB
  • Acceso al software, información las 24 hrs y desde cualquier lugar. Análisis de requerimientos, desarrollo ágil, implementación y capacitación, componen el proceso.
  • VANGUARDIA
  • Desarrollamos la interfaz de usuario con tecnologías que están a la vanguardia, dinamismo con javascript (frameworks: AngularJs, Jquery), estándares web con HTML5-CSS3, uso de preprocesadores de CSS (less) y frameworks visuales como Bootstrap.
  • DOMINIOS
  • Asesoramos a nuestros clientes en la compra de dominios y realizamos su gestión.

Diseño Web

  • DISEÑO WEB
  • En nuestros proyectos realizamos el diseño adecuado para cada cliente, siempre enfocándonos en la usabilidad del sitio para generar una experiencia agradable.
  • MAQUETADO WEB
  • Brindamos servicios de maquetación web a clientes cuyos sitios web requieren de retoques de estilo para mejorar la adaptabilidad y/o usabilidad.
  • WEB RESPONSIVE
  • Cada proyecto realizado por Gilums se adapta a cualquier dispositivo, su página web se podrá visualizar de la misma manera en una tablet, en un dispositivo mobile, en una notebooks y en cualquier monitor.
  • MOCKUPS
  • Antes de poder realizar el diseño del proyecto, realizamos varios diseños digitales para que el cliente pueda visualizar una posible versión final de su sitio web.

Mobile

  • Nativo / Hibrido
  • Podemos crear aplicación para que esté al alcance de todos, se pueden hacer de forma nativa o no nativa pero que funciona de la misma forma. Nuestras aplicaciones se pueden visualizar en cualquier dispositivo mobile.

SOBRE NOSOTROS

Gilums nace de una noche de programación sumado de unas cuantas copas.
La palabra "Gilums" proviene de un insulto generalizado hacia un grupo de personas tercas, tontas y porfiadas.
Este grupo tan peculiar se formó gracias a los trabajo que se realizaron en las diferentes tareas para el Tecnólogo Informático de la Facultad de Ingeniería.
Trabajar en equipo es la esencia para el buen ambiente laboral y la calidad de nuestros servicios.
